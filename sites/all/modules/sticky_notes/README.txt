
README Sticky Notes Drupal Module

# Description

This module adds a new content type sticky note. You can attach sticky notes to all pages in your Drupal site.

# Dependencies
Drupal 6.x
Modal Frame API 6.x-1.6

# Installation
Get and install Modal Frame API: http://drupal.org/project/modalframe
Checkout the module from github: git clone http://github.com/berliner/sticky_notes.git
Copy the folder to your modules directory, normally under sites/all/modules.
Go to admin/build/modules and enable the sticky notes module.
Go to admin/user/permissions and set the permissions.
Go to admin/settings/sticky_notes and configure the module.
That's it.
