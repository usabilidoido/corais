URL paths that contain non-ASCII characters

If your Drupal URL paths contain non-ASCII characters, you may have to tweak your locate settings on the server in order to ensure the URL paths get correctly translated into directory paths in the file system. You can also turn off the ASCII filter in the Boost Advanced Settings. OR install the Transliteration module to help fix the characters.
