The following contributed modules are supported:

  • Automatic Nodetitles
  • Domain Access
  • Location
  • Modr8 (Moderate)
  • Organic Groups
  • Ubercart

Automatic Nodetitles

The Automatic Nodetitles module is fully supported. Node import respects following settings:

Disabled
The Title field is required and mappable.
Automatically generate the title and hide the title field
The Title field is not required and not mappable.
Automatically generate the title if the title field is left empty
The Title field is not required and mappable.

Domain Access

The Domain Access module is fully supported.

Content fields

Content fieldFormat
Send to all affliatesBoolean
Publish toAllowed values
You can use either the Id or the Site name from the Domain list. To publish to the primary domain use -1.

Organic groups

Content types can be a Group node or a Standard group post. The available content fields differ between the two types. No content fields are added for content types that are none of the two.
