Core Drupal 6.x supported modules.

  • Blog
  • Book
  • Comment
  • Node

Blog

The Blog module is fully supported.

Content types

You can import the Blog entry content type.

Content fields

The Blog module does not add any extra content fields.

Book

The Book module is fully supported.

Content types

You can import the Book page content type.

Content fields

The Book module adds following content fields for content types that can be added to a book:

Content fieldFormat
Book outline
BookBook reference
Parent itemBook page reference
WeightWeight (-15:15)

Comment

Node

The Node module is fully supported.

Content types

You can import any of the content types defined on the Content types administration page such as Page or Story.

Content fields

The Node module adds following content fields for all content types:
