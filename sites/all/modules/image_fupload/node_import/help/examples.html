Basic node content example

This is an example that imports just the basic node content fields.

Assumptions

  • You have defined a Story content type,
  • there is a user webmaster with UID equal to 1 and e-mail address webmaster@example.org,
  • you have downloaded bird-houses.csv.

New import

New import
Start a new import.
Press Next.
Select content type
Select Story content type.
Press Next.
Select file
Upload bird-houses.csv.
It will be selected after pressing Upload file.
Press Next.
Select file options
Make sure Comma Separated Values is selected.
Press Next.
Map file columns
The mapping is autodetected because of the carefully selected file columns titles.
Press Next.
Set import options
You can set the date format for the Authored on content field. As all dates are in ISO date format in the file, you do not need to change this for this example.
Press Next.
Set default values
You can set the default values for all content fields. You will want to change the Input format to Full HTML.
Press Next.
Preview import
You see a preview of all imported records. You should not see any errors (marked in red).
Press Next.
Start import
The Name is already set to the name of the CSV file. The name can be used to keep track of different imports.
Press Start import.

You are now redirected to a page where you can see the progress of the import. All 16 rows should import without errors.
