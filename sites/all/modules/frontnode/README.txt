Front Node
-----------

A small module which allows users to set node as a front page.

Install
-------

To install, place the entire frontnode folder into you modules directory.
Go to Administer -> Site building -> Modules and enable Front Node module.

Usage
-----

After installation of the module there is a new node-edit form option "Use node as front page".
After the option will be checked , the node is displayed on the front page.
