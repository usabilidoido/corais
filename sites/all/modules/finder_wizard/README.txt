INTRODUCTION
------------
Splits Finder elements onto seperate screens.
Maintainer: Daniel Braksator (http://drupal.org/user/134005)
Project page: http://drupal.org/project/finder_wizard.


INSTALLATION
------------
1. Download the Finder module and follow it's README.txt http://drupal.org/project/finder.
2. Copy finder_wizard folder to modules (usually sites/all/modules) directory.
3. At admin/build/modules enable the Finder Wizard module.


CONFIGURATION
-------------
1. Enable permissions at admin/user/permissions.
2. Create finders and their form elements at admin/build/finder.
3. Activate the finder's block or navigate to the finder's path.
